**Question:** Place the acronym of the Python interpreter used in INTERACTIVE mode in the correct order.

**Answer:**
- Step 1: **Read**
- Step 2: **Evaluate**
- Step 3: **Print**
- Step 4: **Loop**

> The correct answer is **REPL** — Read → Evaluate → Print → Loop
