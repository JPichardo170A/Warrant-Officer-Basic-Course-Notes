# Elastic Compute Cloud (EC2):

	A web service that provides secured, resizable compute capacity in the cloud. 
	Run Virtual machines (called Instances) on demand just like physical computers. 

	* Scalable

# Azure Virtual Machines same concept of AWS Cloud:

	On demand, scalable, computing resources in the Azure env. 
	Run applications, host windows, or Linux workloads, perform dev, testing, or product tasking. 

	* Offer on demand and scalable as well. 

# Cloud Storage Security: 

	Key Pillars of Cloud Storage Security:
		a. Encryption
		b. Access Control
		c. Monitoring and Auditing
		d. Back up and Recovery 

 	* Encrypted by default from vendor upon spun up. 
	
	S3: AWS Object storage services used for storing and retrieving any amount of data at any time.
		Encrypted at rest: SSE-S3   managed by AWS
				   SSE-KMS  Customer Managed Keys Via (AWS-KMS)
				   SSE-C    Customer provided Keys

		Encrypted in transit: HTTPS/TLS

	* Cloud Storage Security: Azure Blob: Microsoft objects storage solutions for unstructured data like documents, images, videos, and backups.
