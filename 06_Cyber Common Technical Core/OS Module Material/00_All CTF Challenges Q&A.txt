	-> Step 1: # Setup
$startZip = "Omega1000.zip"
$workingDir = "Omega1000.Unzipped"
$layer = 1

# Create working directory
if (-not (Test-Path $workingDir)) {
    New-Item -ItemType Directory -Path $workingDir | Out-Null
}

# Copy initial zip
Copy-Item $startZip "$workingDir\Layer_$layer.zip"

# Loop through nested zips
while ($true) {
    $zipPath = "$workingDir\Layer_$layer.zip"
    $extractPath = "$workingDir\Layer_$layer"

    Expand-Archive -Path $zipPath -DestinationPath $extractPath -Force
    Write-Host "📂 Unzipped Layer $layer"

    # Look for next zip
    $nextZip = Get-ChildItem -Path $extractPath -Filter *.zip -Recurse | Select-Object -First 1

    if ($nextZip) {
        $layer++
        Copy-Item $nextZip.FullName "$workingDir\Layer_$layer.zip"
    } else {
        Write-Host "✅ Final layer reached at Layer $layer"
        break
    }
}

# Read contents of final layer
$finalFiles = Get-ChildItem -Path $extractPath -File -Recurse

foreach ($file in $finalFiles) {
    Write-Host "`n📖 Reading file: $($file.Name)"
    Get-Content $file.FullName
}

2. How many bits are in a nibble, and a byte?

1. [Definition of a Bit]
   - A bit (binary digit) is the smallest unit of data in computing.
   - It can hold a value of either 0 or 1.

2. [What is a Nibble?]
   - A nibble is defined as **4 bits**.
   - It can represent 2⁴ = 16 possible values (0 to 15 in decimal, 0x0 to 0xF in hex).
   - Commonly used in hexadecimal representation, where each hex digit maps to one nibble.

3. [What is a Byte?]
   - A byte is defined as **8 bits**.
   - It can represent 2⁸ = 256 possible values (0 to 255 in decimal, 0x00 to 0xFF in hex).
   - Standard unit for storing characters, instructions, and memory addressing in most architectures.

4. [Why These Sizes?]
   - Early computing systems standardized 8-bit bytes for compatibility and memory alignment.
   - Nibbles are useful for splitting bytes in half, especially in low-level operations like bit masking, hex parsing, and binary encoding.

[Historical Note]
- The term "nibble" is a play on "byte" — a nibble is half a byte, just like a nibble is a small bite of food.

[Final Flag]
4,8
