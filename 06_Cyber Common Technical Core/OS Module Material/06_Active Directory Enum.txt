=========================================
 WINDOWS ACTIVE DIRECTORY ENUMERATION Q&A
=========================================

1. What is the database that is used to connect users with network resources?
   -> Step 1: Active Directory is the centralized database for connecting users with resources.
   -> Answer: Active Directory

2. What are all things that are in Active Directory stored as?
   -> Step 1: In Active Directory, every element (users, computers, printers, etc.) is stored as an object.
   -> Answer: object

3. What is the Active Directory component which contains formal definitions of every object class that can be created in an Active Directory forest?
   -> Step 1: Active Directory Schema defines all object classes and their attributes.
   -> Answer: Active Directory Schema

4. What protocol is used when accessing and updating the Active Directory?
   -> Step 1: LDAP (Lightweight Directory Access Protocol) is the standard protocol for AD communication.
   -> Answer: LDAP

5. From an Offensive perspective, what type of account is usually the main target?
   -> Step 1: Administrator accounts have the highest privileges, making them prime targets.
   -> Answer: Administrator

6. If an account has been inactive for a substantial amount of time what should administrators do to the account?
   -> Step 1: Inactive accounts should be disabled to reduce attack surface.
   -> Answer: Disable

7. What is the basic PowerShell cmdlet used to enumerate users?
   -> Step 1: Use Get-ADUser to enumerate users in Active Directory.
   -> Answer: Get-ADUser

8. What is the suite of tools used in CLI to enumerate users across the network?
   -> Step 1: Domain Services provides the CLI tools.
   -> Answer: Domain Services

9. What is the domain portion of the following SID: S-1-5-21-1004336348-1177238915-682003330-1000
   -> Step 1: Extract domain portion from the SID.
   -> Answer: 21-1004336348-1177238915-682003330

10. What PowerShell command will list domain groups?
    -> Step 1: Run the AD cmdlet to enumerate groups.
    -> Command: Get-ADGroup -Filter *
    -> Answer: Get-ADGroup -Filter *

11. What PowerShell command will list all users and their default properties? The flag is the full command with arguments.
    -> Step 1: Enumerate users with their default properties.
    -> Command: Get-ADUser -Filter 'Name -like "*"'
    -> Answer: Get-ADUser -Filter 'Name -like "*"'

12. What PowerShell command will allow you to search Active Directory accounts for expired accounts without having to create a filter?
    -> Step 1: Use built-in cmdlet for expired accounts.
    -> Answer: search-ADAccount

13. Find the expired accounts that aren't disabled. List the last names in Alphabetical Order, separated with a comma, and no space between.
    -> Step 1: Command -> get-aduser -filter {Enabled -eq "FALSE"} -properties name, enabled
    -> Step 2: Command -> get-localuser | Where-Object { $_.AccountExpires -le (Get-Date)}
    -> Step 3: Command -> ( Get-ADUser -Filter * -Properties Surname,Enabled,AccountExpirationDate |
       Where-Object {
         $_.Enabled -eq $true -and
         $_.AccountExpirationDate -and
         $_.AccountExpirationDate -le (Get-Date) -and
         $_.Surname
       } |
       Select-Object -ExpandProperty Surname |
       Sort-Object -Unique
    ) -join ','
    -> Answer: Krause,Page

14. Find the unprofessional email addresses. List the email's domain.
    -> Step 1: Get-ADUser -Filter * -Properties Mail,UserPrincipalName |
               Where-Object { $_.Mail -or $_.UserPrincipalName } |
               Select-Object Name, SamAccountName, Mail, UserPrincipalName | Format-Table -AutoSize
       Output shows: @mail.mil and ashleymadison.com
    -> Step 2: Get-ADUser -Filter * -Properties Mail |
               Where-Object { $_.Mail -and ($_.Mail -notlike '*.mil@mail.mil') } |
               Select-Object Name,Mail
    -> Step 3: Extract just the domains.
               Command: Get-ADUser -Filter * -Properties Mail |
                        Where-Object { $_.Mail -and ($_.Mail -notlike '*.mil@mail.mil') } |
                        ForEach-Object { $_.Mail.Split('@')[-1].ToLower() } | Sort-Object -Unique
    -> Step 4: Join results.
               Command: ( Get-ADUser -Filter * -Properties Mail |
                          Where-Object { $_.Mail -and ($_.Mail -notlike '*.mil@mail.mil') } |
                          ForEach-Object { $_.Mail.Split('@')[-1].ToLower() } |
                          Sort-Object -Unique
                        ) -join ','
    -> Answer: ashleymadison.com

15. The flag is the unprofessionally-named file located somewhere on the Warrior Share.

16. The flag is the name of the file where someone is requesting modified access rights.

18. Find accounts with unprofessional information in the description.
    -> Step 1: Import AD module.
               Command: Import-Module ActiveDirectory
    -> Step 2: List users with descriptions.
               Command: Get-ADUser -Filter * -Properties Description | Select-Object Name,Description
    -> Step 3: Identify flagged accounts: Jimenez, Brandywine, Ibarra, Lee
    -> Step 4: Extract last names alphabetically.
    -> Answer: Brandywine,Ibarra,Jimenez,Lee

19. Find 2 accounts with passwords that never expire and 1 with reversible encryption.
    -> Step 1: Get all users with extra properties.
               Command: Get-ADUser -Filter * -Properties PasswordNeverExpires,UserAccountControl
    -> Step 2: Format output.
               Command: Get-ADUser -Filter * -Properties PasswordNeverExpires,UserAccountControl |
                        Format-Table Name, SamAccountName, PasswordNeverExpires, UserAccountControl
    -> Step 3: (Optional) Filter accounts with "Password does not expire" flag.
               Command: Get-ADUser -Filter {UserAccountControl -band 0x10000} -Properties PasswordNeverExpires,UserAccountControl |
                        Format-Table Name, SamAccountName, PasswordNeverExpires, UserAccountControl
    -> Answer: Brandywine,Ibarra,Sanchez

23. How many users are members of the Domain Admins group? (No sub-groups)
    -> Step 1: Import Active Directory module.
               Command: Import-Module ActiveDirectory
    -> Step 2: Get members of Domain Admins.
               Command: Get-ADGroupMember -Identity "Domain Admins" | Where-Object { $_.objectClass -eq "user" }
    -> Step 3: Count the users.
    -> Answer: 1

24. How many total users are members of the Domain Admins group?
    -> Step 1: Get all Domain Admins.
               Command: Get-ADGroupMember -Identity "Domain Admins"
    -> Step 2: Count users.
               Command: (Get-ADGroupMember -Identity "Domain Admins" | Where-Object { $_.objectClass -eq "user" }).Count
    -> Answer: 14
