## 🎯 Purpose

This repository serves as a centralized, structured archive built to:

- 📝 Consolidate course materials, notes, and personal reflections
- 🔍 Highlight key lessons learned and practical applications
- 🔗 Provide references and resources used throughout the course
- 🔄 Serve as a living document that grows with continued learning and development

---

## 🚀 How to Use

Each module folder contains:
- A `README.md` with a full file listing and topic descriptions
- Individual topic files organized by day or concept
- Code examples, step-by-step procedures, and key takeaways

> **New here?** Start at [01_Python](01_Python/) or [02_PowerShell](02_PowerShell/) for the core technical content, or jump directly to the module most relevant to your work.
